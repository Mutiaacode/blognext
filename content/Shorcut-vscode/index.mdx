Visual Studio Code (VSCode) telah menjadi pilihan utama banyak pengembang berkat antarmuka pengguna yang intuitif dan fitur-fitur canggih. Dalam panduan ini, kita akan menjelajahi dunia pintasan keyboard yang memungkinkan pengembangan cepat dan efisien menggunakan VSCode. Mulai dari navigasi file yang lebih cepat hingga refactor kode dengan mudah, setiap shortcut akan diuraikan dengan jelas, dibarengi dengan contoh penggunaan langsung dalam skenario pengembangan nyata. Selain pintasan, artikel ini juga akan mengungkapkan fitur-fitur tersembunyi dan ekstensi yang dapat meningkatkan pengalaman pengembangan Anda. Bersiaplah untuk mengoptimalkan waktu dan produktivitas Anda dengan panduan lengkap ini untuk memaksimalkan potensi penuh Visual Studio Code!

Berikut adalah daftar beberapa pintasan keyboard (shortcut) yang berguna di Visual Studio Code (VSCode). Pastikan Anda mengonfirmasi bahwa shortcut ini sesuai dengan versi terbaru VSCode karena ada kemungkinan perubahan pada versi yang lebih baru.

### Navigasi dan Manajemen File:
- Ctrl + P: Buka Quick Open (Cari file).
- Ctrl + Shift + N: Buka jendela Explorer.
- Ctrl + B: Sembunyikan/Tampilkan panel samping.

### Pintasan Editor:
- Ctrl + /: Komentari atau nonaktifkan baris.
- Alt + ↑ / ↓: Pindah baris ke atas atau ke bawah.
- Ctrl + Shift + K: Hapus baris.
- Ctrl + D: Pilih seluruh kemunculan kata yang dipilih.
- Alt + Click: Pilih area teks dengan klik.
- Ctrl + Space: Trigger suggestion (pilihan otomatis) saat menulis kode.
- Ctrl + Shift + L: Pilih semua kemunculan kata saat mengetik.
- Ctrl + Shift + \: Pergi ke kurung kurawal yang sesuai.

### Pintasan Navigasi dalam File dan Antarmuka:
- Ctrl + Tab: Navigasi tab yang terbuka.
- Ctrl + Shift + Tab: Navigasi tab yang terbuka secara terbalik.
- Ctrl + F4: Menutup editor yang aktif.

### Pintasan Debugging:
- F5: Mulai debugging.
- F110 / F11: Langkah satu per satu dalam debugging.

### Pintasan Umum:
- Ctrl + Z: Batalkan perubahan.
- Ctrl + Shift + Z: Redo perubahan.
- Ctrl + X: Potong.
- Ctrl + C: Salin.
- Ctrl + V: Tempel.

Ini hanya sebagian kecil dari semua pintasan yang tersedia di VSCode. Anda dapat menyesuaikan preferensi Anda dan menemukan pintasan tambahan di dokumentasi resmi VSCode atau dengan mengeksplorasi menu Command Palette (Ctrl + Shift + P) yang memungkinkan Anda mengeksplorasi dan menjalankan banyak fungsi dan perintah.
