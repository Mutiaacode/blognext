JavaScript (JS) adalah salah satu bahasa pemrograman yang paling populer di dunia web. Sebagai bahasa scripting yang berjalan di sisi klien (client-side) pada browser, JavaScript memberikan kekuatan dan interaktivitas pada situs web. Artikel ini akan membahas dasar-dasar programming dengan JavaScript, memberikan wawasan tentang sintaksis dasar, variabel, pengulangan, dan fungsi.


## Apa Itu JavaScript?
JavaScript adalah bahasa pemrograman yang awalnya dikembangkan untuk meningkatkan interaktivitas pada halaman web. Saat ini, JavaScript digunakan di berbagai tempat, termasuk pengembangan web, server-side programming, dan pengembangan aplikasi mobile.

## Dasar-Dasar JavaScript:
1. Variabel:
Variabel digunakan untuk menyimpan nilai. Deklarasikan variabel dengan kata kunci var, let, atau const. Contoh:

```javascript showLineNumbers {1} /science/ title="javascript" caption="variabel"
let angka = 10;
```

## 2. Tipe Data:
JavaScript memiliki beberapa tipe data seperti String, Number, Boolean, Array, dan Object. Pahami cara menggunakan setiap tipe data.

```javascript showLineNumbers {1-4} /science/ title="javascript" caption="Tipe Data"
let nama = "John";
let umur = 25;
let isActive = true;
let daftarAngka = [1, 2, 3, 4, 5];
```

## 3. Pengulangan:
Gunakan pernyataan for atau while untuk mengulang tugas tertentu.

```javascript showLineNumbers {1-3} /science/ title="javascript" caption="Pengulangan"
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

## 4. Percabangan:
Gunakan pernyataan if, else if, dan else untuk menjalankan blok kode berdasarkan kondisi tertentu.

```javascript showLineNumbers {1-9} /science/ title="javascript" caption="Percabangan"
flet nilai = 75;

if (nilai >= 80) {
  console.log("Excellent!");
} else if (nilai >= 60) {
  console.log("Good job!");
} else {
  console.log("Keep trying!");
}
```

## 5. Fungsi:
Fungsi digunakan untuk mengeksekusi blok kode tertentu. Anda dapat mendefinisikan fungsi dan memanggilnya.

```javascript showLineNumbers {1-6} /science/ title="javascript" caption="Pengulangan"
function tambah(a, b) {
  return a + b;
}

let hasil = tambah(3, 5);
console.log(hasil);  // Output: 8
```

## Sumber Daya dan Lanjutan:
JavaScript memiliki ekosistem yang kaya dengan banyak sumber daya online untuk belajar lebih lanjut. Anda dapat memanfaatkan dokumentasi resmi di MDN Web Docs dan mencoba platform pembelajaran seperti Codecademy atau freeCodeCamp.

Dengan memahami dasar-dasar JavaScript, Anda dapat membangun landasan yang kuat untuk memperluas keterampilan programming Anda. Ingatlah untuk terus berlatih dan mencoba proyek-proyek kecil. Selamat belajar!
