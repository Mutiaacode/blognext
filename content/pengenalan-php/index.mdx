PHP (Hypertext Preprocessor) adalah bahasa pemrograman server-side yang sangat populer, khususnya untuk pengembangan web dinamis dan aplikasi. Dengan fokus pada kemudahan penggunaan dan kemampuan integrasi yang kuat, PHP telah menjadi salah satu alat yang paling penting dalam dunia pengembangan web. Artikel ini akan membahas dasar-dasar programming dengan PHP, mencakup variabel, kontrol alur, fungsi, dan database.

## Apa Itu PHP?
PHP adalah bahasa scripting server-side yang dirancang untuk membuat halaman web dinamis. PHP dijalankan di server web dan menghasilkan HTML yang dikirim ke browser pengguna. Dengan kata lain, PHP memungkinkan Anda membuat halaman web yang dapat berinteraksi dengan pengguna dan mengakses database.

## Dasar-Dasar PHP:
1. Variabel dan Tipe Data:
Deklarasikan variabel dengan awalan dollar sign ($) dan tentukan tipe data secara otomatis.

## 2. Pengulangan dan Percabangan:
Gunakan pernyataan for, while, dan if untuk mengontrol alur program.

```php showLineNumbers {1-9} /science/ title="javascript" caption="Pengulangan Dan Percabangan"
for ($i = 0; $i < 5; $i++) {
  echo $i;
}

if ($umur >= 18) {
  echo "Sudah dewasa!";
} else {
  echo "Masih di bawah umur.";
}
```
## 3. Fungsi:
Tentukan dan panggil fungsi untuk memecah program menjadi bagian-bagian yang lebih kecil

```php showLineNumbers {1-6} /science/ title="javascript" caption="Fungsi"
function tambah($a, $b) {
  return $a + $b;
}

$hasil = tambah(3, 5);
echo $hasil;  // Output: 8
```
## 4. Koneksi Database:
PHP sering digunakan untuk berinteraksi dengan database. Gunakan ekstensi seperti MySQLi atau PDO untuk menghubungkan PHP dengan database MySQL atau jenis database lainnya.

```php showLineNumbers {1-7} /science/ title="javascript" caption="Koneksi Ke Database"
$koneksi = new mysqli("localhost", "username", "password", "database");

if ($koneksi->connect_error) {
  die("Koneksi gagal: " . $koneksi->connect_error);
}

// Lakukan operasi database di sini...
```
